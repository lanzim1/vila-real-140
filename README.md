# Vila Real 140 — Sistema de Gestão de Condomínio

## Como rodar localmente
npm install
npm run dev

## Como publicar (deploy)
1. Suba esta pasta para um repositório no GitHub
2. Conecte o repositório na Vercel (vercel.com) — ela detecta o Vite automaticamente
3. Clique em "Deploy"
